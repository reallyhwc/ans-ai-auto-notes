**规则：绘图展示时优先使用 Mermaid 渲染图**

Why: 用户明确指出不要单纯用 ASCII 绘图风格；overview.html 已集成 Mermaid 运行时，能渲染为可视化 SVG，效果远好于 ASCII 框图。此规则已在 `CLAUDE.md`「笔记风格规则」第 2 条同步明文化。

How to apply:
- 流程图 → `flowchart TD/LR`
- 时序图 → `sequenceDiagram`
- 架构图 → `flowchart TB` + `subgraph`
- ASCII 框图仅在 Mermaid 不适用的场景使用（如非常简单的树状列表、需要保留 monospace 对齐的代码示意）
- 参考用法：项目 `kb/` 下已有 20+ 文件使用 mermaid，用 `grep -rl '^\`\`\`mermaid' kb/` 即可定位最新示例（具体文件路径会随重构变化，不在此列举）
