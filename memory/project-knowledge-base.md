项目 `ans-ai-auto-notes` 是一个 AI 对话驱动的个人知识库模板/脚手架。
采用 git + markdown 形式，主题分类（技术/实战/读书笔记）和时间线按周归档双轨并行。
入口文件：INDEX.md（总目录）、overview.html（可视化导览，支持分类浏览、搜索、词云、Mermaid 渲染）。

**更新策略**：混合模式 — 小知识点自动追加，大改动（拆分/合并/重组）主动提案待确认。

**文件拆分阈值**（已写入 CLAUDE.md，任意两条触发）：
- 行数 > 1000 关注，> 1500 必须拆
- `##` 级章节 > 7-8
- 覆盖 3+ 个可独立成文的方向

**Why:** 用户希望在不打断对话流的情况下自动积累结构化知识，同时保留对重大改动的掌控权。

**How to apply:** 每次对话后自动提取知识点写入对应文件；发现文件过大或结构需调整时主动提案。笔记风格必须是带 Demo 的对话式讲解，不是教科书定义。
