# CNN (卷积神经网络)

## 2026-05-03 - CNN 核心概念入门
> 来源: 对话 2026-05-03
> 关联: [[../../读书笔记/我看见的世界]]、[[./rnn]]、[[./transformer]]

### 定位

专门处理图像/空间数据的神经网络。核心操作是"用一个小窗口滑过整张图，找特征"。

### 核心机制

1. **卷积（Convolution）**：用一个小矩阵（卷积核/滤波器，如 3×3）在输入图像上滑动，每个位置做"逐元素相乘再求和"，输出一个特征图（Feature Map）。不同卷积核能检测不同特征——边缘、纹理、颜色变化等。

2. **池化（Pooling）**：缩小特征图的尺寸（如 2×2 取最大值 Max Pooling），减少计算量，同时让检测对位置不那么敏感。

3. **层级特征提取**：底层卷积学边缘/颜色 → 中层学形状/纹理 → 高层学语义（"眼睛"、"轮子"等）。

### 经典结构

```
输入图 → 卷积层(提特征) → 池化层(降维) → 卷积层 → 池化层 → ... → 全连接层 → 输出
```

### 关键特点

- **天然并行**：卷积操作不依赖顺序
- **参数共享**：同一个卷积核扫全图，参数量少
- **平移不变性**：特征无论出现在图的哪个位置都能检测到

### 与 ImageNet 的关系

李飞飞创建的 ImageNet（1400万张标注图片）在 2012 年被 AlexNet（CNN 架构）大幅刷榜，直接引爆了这轮深度学习浪潮。
