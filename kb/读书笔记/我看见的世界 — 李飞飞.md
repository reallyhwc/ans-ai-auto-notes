# 我看见的世界 — 李飞飞

> 最后整理: 2026-05-04 | 来源: 多轮对话

## AI 基础概念扫盲

阅读过程中对 CNN、RNN、Transformer 三大核心概念不清楚，进行了系统梳理。


### CNN（卷积神经网络）
- **分层特征涌现**：浅层→中层→深层的分层结构不需要逐层人工标注，ImageNet 只有图片级标签，分层是通过反向传播自然涌现的。证据：Zeiler & Fergus 2014 反卷积可视化实验。→ [CNN 3.4](../技术/AI/基础/CNN（卷积神经网络）.md#3-4-分层特征不需要人工标注-是自己-涌现-的)
- **层1可解释性**：层1特征"能定义但不好用自然语言描述"——数学精确（"3×3 窗口内从左到右亮度递增"），但日常语言没造这个词。从 Layer 1 到 Layer N 是数学语言→自然语言的渐变。→ [CNN 3.5](../技术/AI/基础/CNN（卷积神经网络）.md#3-5-层1到底-识别-出了什么-数学语言-vs-自然语言)
- **ImageNet 历史时刻**：李飞飞创建的 ImageNet（1400 万张标注图片）与 AlexNet（CNN 架构）在 2012 年结合，正确率从 ~70% 跳到 ~85%，引爆了深度学习浪潮。→ [CNN 6](../技术/AI/基础/CNN（卷积神经网络）.md#6-cnn-与-imagenet-的历史时刻)


### RNN（循环神经网络）
- **核心机制**：RNN 没有卷积核——靠三套共享权重矩阵（W_xh/W_hh/W_hy）在时间维度上反复使用。训练用 BPTT（沿时间反向传播），与 CNN 一样靠反向传播+梯度下降。→ [RNN](../技术/AI/基础/RNN（循环神经网络）.md)
- **梯度消失**：朴素 RNN 处理长文本时梯度消失，催生了 LSTM/GRU。


### Transformer
- **核心创新**：自注意力 Q/K/V 机制——抛弃 RNN 的串行传递，所有词并行互相"看"，任意两词 O(1) 关联。→ [Transformer](../技术/AI/基础/Transformer.md)
- **历史地位**：2017 年提出后逐步取代 RNN，Decoder-only 架构是现代大模型（GPT、Claude）的底座。


### LLM（大语言模型）
- **本质**：Decoder-only Transformer 做到极致——规模引发涌现能力。→ [LLM](../技术/AI/大模型/LLM（大语言模型）.md)
- **Prompt 与 RAG**：Prompt 工程（Zero/Few-shot/CoT）、RAG 检索增强生成、向量数据库与 Embedding 模型。→ [Prompt与RAG](<../技术/AI/大模型/Prompt 与 RAG.md>)
- **Agent 与生态**：Agent 循环（思考→行动→观察）、MCP 协议、AI 工具五层分类、Claude Code 工作原理。→ [Agent与MCP](<../技术/AI/大模型/Agent 与 MCP.md>)
